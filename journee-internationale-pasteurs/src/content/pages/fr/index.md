# Guide de Maintenance

Bienvenue dans la documentation technique du site de la Journée Internationale des Pasteurs.

## Architecture

Le site utilise une architecture hybride :
- **Site Vitrine** : Pages gérées via Keystatic dans `src/content/pages`.
- **Documentation** : Cette section, gérée par Starlight dans `src/content/docs`.

## Comment mettre à jour le contenu ?

Accédez à l'interface d'administration via `/keystatic`.
